# Theme Catalog

SlideForge のテーマ説明一覧です。テーマは情報構造を変えず、見た目とトーンだけを切り替えるために使います。すべてのテーマは、デジタル庁デザインシステムの考え方を参考に、可読性、情報階層、余白、アクセシビリティを共通の前提にします。

## business

経営層、提案資料、業務改善、合意形成向けの標準テーマです。白から淡いブルーグレーを基調に、落ち着いたカードと控えめな影で構成します。

向いている用途:

- 稟議、提案、ロードマップ
- KPI 報告
- 業務フローや比較表

## engineer

技術説明、アーキテクチャ、コード、運用設計向けのテーマです。コントラストをやや強くし、構造図やコードブロックが見やすいようにします。

向いている用途:

- 技術選定
- システム構成
- 障害報告、運用改善

## fancy

イベント登壇、コンセプト資料、印象づけが必要な企画資料向けのテーマです。視覚的な勢いを持たせつつ、スライド role は標準構造を維持します。

向いている用途:

- ピッチ
- コンセプト紹介
- クリエイティブ寄りの企画

## pop

SNS、ライトな紹介、キャラクターや動画素材との相性を重視したテーマです。明るい色面と大きめの要素で、短時間でも意味が伝わる構成にします。

向いている用途:

- ショート動画の背景スライド
- サービス紹介
- 初学者向け説明

## mobile

1080 x 1920 px の縦型テーマです。スマホ視聴を前提に、1 枚あたりの情報量を少なくし、タイトルと要点を大きく見せます。

向いている用途:

- 縦型ショート動画
- SNS ストーリー
- モバイル閲覧用の簡易資料
