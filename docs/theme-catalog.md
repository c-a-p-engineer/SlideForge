# Theme Catalog

SlideForge のテーマ説明一覧です。テーマは情報構造を変えず、見た目とトーンだけを切り替えるために使います。すべてのテーマは、デジタル庁デザインシステムの考え方を参考に、可読性、情報階層、余白、アクセシビリティを共通の前提にします。

現在同梱しているテーマは `light` と `dark` の 2 種類です。どちらも 1920 x 1080 px の横型スライドを前提にしています。

## light

白から淡いグレーを基調にした標準テーマです。提案資料、社内説明、意思決定資料など、印刷・共有・レビューのどれにも使いやすい見た目です。

向いている用途:

- 提案資料、稟議、合意形成
- KPI 報告、ロードマップ
- 比較表、ガントチャート、問い合わせ先

デザイン方針:

- 背景は明るく、本文色は黒に近い色で読みやすくする
- 重要数値は青、導線や矢印は緑寄りのアクセントで分ける
- カードや説明ボックスは軽い境界線と控えめな影に留める

## dark

深い紺色ベースのテーマです。画面共有、動画素材、テック寄りの提案資料でコントラストを強く出したいときに使います。

向いている用途:

- 技術説明、サービス構成図
- 登壇資料、動画背景、画面共有
- フロー、システム図、コード、チャート

デザイン方針:

- 背景は深い紺色、文字は白寄りでコントラストを確保する
- セクションや構成図には薄いグリッドと光彩を使い、テック感を足す
- 水色を重要数値、緑を導線や処理フローに使い分ける

## テーマ選択

- 迷ったら `light` を使います。
- プレゼン画面や動画で印象を強めたい場合は `dark` を使います。
- どちらのテーマでも slide type と情報構造は同じにします。テーマごとにページの意味や順序を変えないでください。
